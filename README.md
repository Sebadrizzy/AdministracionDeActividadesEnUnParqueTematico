## 🚀 Instrucciones de Instalación y Ejecución

1. **Requisitos Previos:**
   * Tener instalado **Java JDK 8 o JDK 11** (Oracle o OpenJDK).
   * Un IDE compatible (Netbeans 21, Eclipse o VS Code).

2. **Ejecución:**
   * Clonar este repositorio o extraer el archivo ZIP entregado.
   * Abrir el proyecto en el IDE seleccionado.
   * Ejecutar la clase principal ubicada en `Main.java`.
   * El sistema mostrará un cuadro de diálogo inicial para seleccionar la interfaz.

3. **Archivos Adicionales:**
   * El sistema genera automáticamente un archivo llamado `datos.csv` en la raíz del proyecto para la persistencia de datos. No se requieren bases de datos externas ni la instalación de JARS adicionales.
